4. FUNCIONALIDADES EXISTENTES
───────────────────────────────────────────────────────────────────────────────

► BUSCA BÍBLICA
  • Busca por referência bíblica (ex: "João 3:16", "Sl 23")
  • Busca por capítulo completo (ex: "Gênesis 1")
  • Busca por palavra/frase em toda a Bíblia
  • Suporte a abreviações (ex: "Jo", "Gn", "Rm", "Hb")
  • Filtro por livro bíblico (Antigo/Novo testamento)
  • Busca por frase exata ou todas as palavras

► REFERÊNCIAS CRUZADAS
  • Exibição de referências cruzadas para cada versículo
  • Ordenação por relevância (votos)
  • Destaque para referências mais confirmadas (≥100 votos)
  • Links para visualização de capítulos

► SISTEMA DE FAVORITOS
  • Marcar versículos como favoritos
  • Visualizar lista de favoritos
  • Exportar favoritos para arquivo de texto

► SISTEMA DE MARCADORES
  • Marcar versículos com cores personalizadas
  • Cores disponíveis: Dourado, Verde, Azul, Vermelho, Roxo
  • Marcador customizável (corPicker)

► SISTEMA DE ANOTAÇÕES
  • Anotações pessoais para cada versículo
  • Destaque de texto com cores
  • Anotações inline exibidas abaixo do versículo

► LISTAS TEMÁTICAS
  • Criar listas temáticas personalizadas
  • Adicionar versículos a temas
  • Visualizar e gerenciar listas por tema

► PAINEL DE NOTAS E SERMÕES
  • Editor de notas completo
  • Múltiplas abas de notas
  • Contagem de palavras
  • Auto-salvamento
  • Exportação de notas

► EXPORTAÇÃO
  • Exportar capítulo completo (TXT)
  • Exportar estudo de referências cruzadas (TXT)
  • Exportar favoritos (TXT)
  • Exportar notas individuais (TXT)
  • Copiar para área de transferência

► INTERFACE
  • Design escuro com acentos dourados
  • Sidebar redimensionável
  • Abas múltiplas para pesquisas
  • Sistema de abas para notas
  • Divisor redimensionável entre áreas
  • Popup para detalhes de versículos
  • Tela de boas-vindas com dicas

► OUTROS
  • Sugestões rápidas de versículos
  •Persistência de dados no localStorage
  •Contador de palavras nas notas
  •Suporte a toque/dispositivos móveis
  •Tema visual: "Manuscrito Sagrado" (escuro, dourado, refinado)
